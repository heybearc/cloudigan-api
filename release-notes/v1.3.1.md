# Cloudigan API v1.3.1 Release Notes

**Released:** March 18, 2026

---

## ✨ New Features

### Business vs Personal Product Support

The webhook now intelligently handles both business and personal products, using company names for business subscriptions and customer names for personal subscriptions.

**What you can do:**
- Business products automatically use company name from Stripe checkout
- Personal products use customer name
- Datto RMM sites created with appropriate naming
- Wix CMS tracks both company and customer information
- Emails personalized based on product type

**How it works:**
1. Stripe checkout captures company name in custom field (business products)
2. Webhook extracts company name from `companyname` field
3. Datto site created with company name (business) or customer name (personal)
4. Wix CMS stores both company name and customer name
5. Email greeting personalized: "Hello [Company Name]" or "Hello [Customer Name]"

---

## 🔧 Improvements

### Smart Company Name Extraction

- Extracts company name from Stripe custom field (`companyname`)
- Falls back to customer name for personal products
- Adds `isBusinessProduct` flag to all integrations

### Enhanced Datto Site Creation

- Site name uses company name for business products
- Site description includes business/personal indicator
- Site notes document product type

### Wix CMS Enhancement

- New field: `companyName` (business name or empty)
- New field: `isBusinessProduct` (true/false flag)
- Existing field: `customerName` (display name)
- Confirmation page can differentiate business vs personal

### Email Personalization

- Business products: "Hello [Company Name],"
- Personal products: "Hello [Customer Name],"
- Business products show company name in email body
- Clean, professional formatting for both types

## 💡 What You Need to Know

**No action required** - All changes are backward compatible.

**Business Products:**
- Must have "Company Name" custom field in Stripe pricing table
- Field key must be `companyname`
- Company name will be used for Datto site and displayed in emails

**Personal Products:**
- No company name field required
- Customer name from checkout will be used
- Works exactly as before

**Wix CMS Fields:**
- `companyName` - Business name (empty for personal products)
- `customerName` - Display name (company or customer)
- `isBusinessProduct` - Boolean flag for product type

---

## 📊 Technical Details

**Field Extraction:**
- Stripe custom field key: `companyname`
- Field structure: `session.custom_fields[].text.value`
- Fallback: `session.metadata.company_name`

**Data Flow:**
1. Extract company name from Stripe custom fields
2. Create `displayName` = company name OR customer name
3. Set `isBusinessProduct` flag based on company name presence
4. Pass to all integrations (Datto, Wix CMS, Email)

**Logging:**
- Added debug logging for custom field extraction
- Logs company name, customer name, and product type flag
- Helps troubleshoot field extraction issues
