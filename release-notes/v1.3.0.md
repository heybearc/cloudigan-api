# Cloudigan API v1.3.0 Release Notes

**Released:** March 18, 2026

---

## ✨ New Features

### Wix CMS Integration Restored

The webhook now automatically saves order and download information to your Wix CMS, enabling your custom checkout confirmation page to display download links.

**What you can do:**
- Customers see their download links immediately on the Wix confirmation page
- Order history is automatically stored in Wix CMS
- Download links are available for Windows, macOS, and Linux

**How it works:**
1. Customer completes Stripe checkout
2. Webhook creates Datto RMM site
3. Download links are saved to Wix CMS CustomerDownloads collection
4. Welcome email sent with download links
5. Confirmation page displays links from Wix CMS

### M365 Email Integration

Welcome emails are now sent using your existing M365 OAuth infrastructure instead of SendGrid.

**What you can do:**
- Customers receive professional welcome emails from your domain
- Emails include platform-specific download links
- Fully integrated with your existing M365 setup

---

## 🔧 Improvements

### Correct Download Link Format

Download links now use the proper Datto agent download URL format:
- Windows: `https://vidal.rmm.datto.com/download-agent/windows/{siteUid}`
- macOS: `https://vidal.rmm.datto.com/download-agent/mac/{siteUid}`
- Linux: `https://vidal.rmm.datto.com/download-agent/linux/{siteUid}`

### Wix CMS Field Mapping

Fixed field names to match your Wix CMS collection schema:
- `macOsDownloadLink` - Correctly mapped for confirmation page buttons
- `dattoSiteUid` - Proper site identifier storage
- All fields now populate correctly in Wix CMS

---

## 🐛 Bug Fixes

### Wix Site ID Configuration

Fixed Wix API authentication to use the correct site ID instead of account ID, resolving "MetaSite not found" errors.

### Email Method Call

Corrected M365 email integration to use the proper `sendMail` method matching the M365OAuthMailer interface.

---

## 💡 What You Need to Know

**No action required** - All changes are backward compatible and automatically deployed.

**Environment Variables:**
- `WIX_API_KEY` - Configured via MCP server (4-hour refresh)
- `WIX_SITE_ID` - Set to your Wix site ID
- `M365_CLIENT_ID` - Already configured for email

**Workflow:**
1. Stripe checkout completed
2. Datto site created
3. Wix CMS updated
4. M365 email sent
5. Stripe metadata updated

**Modules Added:**
