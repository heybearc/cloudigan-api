# What's New in Version 1.3.3

**Released:** July 4, 2026

---

## 🐛 Bug Fixes

### Correct payment-failed emails by product

When a subscription renewal fails, customers now receive a **Cloudigan** billing notice with the right product name and a link to the Cloudigan customer portal — not a Chapter Hub message.

**What changed:**
- Complete Package, Home Protect, Business, and support subscriptions get Cloudigan-branded payment-failed emails
- Chapter Hub subscriptions continue to be handled by Chapter Hub only
- Stripe `invoice.payment_failed` events are now processed by this webhook (add the event in Stripe Dashboard if not already enabled)

---

## 💡 What You Need to Know

- **Stripe Dashboard:** Ensure `invoice.payment_failed` is enabled on the `api.cloudigan.net` webhook endpoint
- Webhook endpoint unchanged: `https://api.cloudigan.net/webhook/stripe`
